# cursor-demo

멤버 데이터에서 이메일을 검증·추출하는 유틸리티입니다.

## 설치 및 테스트

```bash
npm test
```

## 릴리스 노트

### v1.0.0

멤버 데이터에서 RFC 5322 기준으로 이메일을 검증·추출하는 유틸리티의 첫 공개 버전입니다.

#### ✨ 기능

- **이메일 검증** — RFC 5322 정규식과 로컬/전체 길이 제한(64/254자)을 적용한 `isValidEmail` 제공
- **이메일 추출** — 멤버 객체 배열에서 이메일 목록을 뽑는 `extractEmails`
- **유효 이메일 필터링** — 검증을 통과한 이메일만 반환하는 `getValidEmails`
- **중복 제거** — 유효 이메일 중복을 제거하고 첫 등장 순서를 유지하는 `uniqueValidEmails`
